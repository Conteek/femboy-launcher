# Femboy Launcher

Femboy Launcher is a desktop Minecraft launcher made for people who want to get into the game quickly without constantly fighting with versions, accounts, Java, mods, and folders.

It brings the most common Minecraft launcher tasks into one place: choose a version, pick an account, install what is missing, launch the game, and keep your modpacks tidy. The launcher supports vanilla Minecraft as well as popular mod loaders, includes account management, and gives you simple tools for working with local modpacks.

## Features

### Version Management

Femboy Launcher can install and launch Minecraft versions directly from the launcher. It shows which versions are already installed and can filter the list so it is easier to find what you need.

Supported version types include:

- Vanilla
- Forge
- Forge + OptiFine
- Fabric
- NeoForge

You can also show only installed versions if you want a cleaner list.

### Accounts

The launcher includes a simple account manager. You can add and switch between:

- Offline accounts
- Ely.by accounts
- Microsoft accounts

Saved accounts are available from the main launcher screen, so switching profiles before launch is quick.

### Modpacks

Femboy Launcher has a built-in local modpack manager. You can create a new modpack from scratch or make a snapshot from selected folders in your current setup.

Modpacks can include details such as:

- Name
- Description
- Author
- Version
- Avatar

You can also edit, delete, import, export, and open modpack folders directly from the launcher.

### Mod Manager

For modded versions and modpacks, the launcher can show the installed mods and their basic metadata when available. It supports common mod formats from Fabric, Quilt, Forge, NeoForge, and legacy Forge mods.

Individual mods can be enabled or disabled without removing them, making it easier to test mod combinations or temporarily turn something off.

### Java and Performance

The launcher can download the required Java runtime when needed, so you do not have to manually hunt for the correct Java version for every Minecraft release or loader.

You can also choose how much RAM Minecraft should use, from lightweight setups to heavier modpacks.

### Quality of Life

Femboy Launcher also includes:

- English and Russian interface languages
- Launcher update notifications
- Discord Rich Presence while browsing the launcher or playing
- A custom desktop window designed around the launcher experience

## Why Use It?

Minecraft setups can get messy fast: different loaders need different versions, modpacks need separate folders, Java versions can break launches, and account switching is annoying when it is hidden behind extra steps.

Femboy Launcher keeps those things close to the surface. It is meant to feel direct: select what you want to play, install it if needed, and press play.
